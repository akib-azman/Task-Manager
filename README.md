# Manager of Tasks

HTML, CSS, and JavaScript were used to create this straightforward task management online application. With persistent storage, this project allows task creation, management, and organization right within the browser.

## Features

Include a title, description, category, and deadline for each new task.
Modify and remove current tasks
Mark assignments as finished.
Sort tasks according to their status (All, Completed, Pending).
Sort assignments according to their due dates.
Organization based on categories
Support for local storage
Simple progress monitoring (statistics on task completion)

## How It Operates

The browser uses `localStorage` to store tasks.
JavaScript modifies the user interface dynamically in response to user input.
There is no use of an external database or backend.

## The Goal of the Project

This project was designed as an educational exercise to comprehend:
DOM manipulation
JavaScript event handling
Utilising local storage
